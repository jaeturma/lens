---
name: implement-work-package
description: Implement one bounded LENS v1.1 work package or one explicitly requested layer.
---

1. Read the specified work package.
2. Read only directly relevant core documents.
3. Inspect the minimum source files.
4. Present a concise plan and affected files.
5. Implement only the requested scope.
6. Preserve school-binding and SQLite-first rules.
7. Run targeted tests and analysis.
8. Review the diff against acceptance criteria.
9. Update implementation notes.
10. Report changed files, migrations, contracts, commands, and risks.

For cross-layer packages, implement only the layer requested in the current session.
