---
name: verify-work-package
description: Verify a completed LENS work package against its acceptance criteria without expanding scope.
---

1. Read the specified work package.
2. Inspect only its changed files and directly related tests.
3. Verify every acceptance criterion.
4. Run targeted tests and static analysis.
5. Fix confirmed defects only.
6. Do not add enhancements or refactor unrelated code.
7. Report pass, fail, remaining risk, and any manual verification required.
