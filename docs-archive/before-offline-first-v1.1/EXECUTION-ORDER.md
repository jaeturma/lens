## Standard Workflow

```text
/implement-work-package docs/work-packages/phase-XX/WP-XX-XX-task.md
```

Then:

```text
/verify-work-package docs/work-packages/phase-XX/WP-XX-XX-task.md
```

Commit the task and run `/clear` before the next unrelated package.

For a package spanning Laravel and Flutter, complete Laravel first in one session, clear, then complete Flutter using the documented API contract.
