# LENS Project Context

## Purpose

LENS is a Digital School System intended to connect school operations, learners, parents or guardians, teachers, guidance personnel, and administrators through a Laravel API and Flutter Android application.

## Current Technology

- Backend: Laravel 13
- Database: MySQL
- Mobile: Flutter for Android
- Authentication: Laravel Sanctum unless an existing implementation is already approved
- Backend tests: Pest

## Initial Delivery Principle

Build the smallest reliable version of each module first. Avoid combining attendance, discipline, guidance, announcements, grades, payments, and notifications into one work package.

## Suggested Initial Mobile Modules

1. Authentication and account recovery
2. Learner profile
3. Attendance history
4. Parent or guardian linked learner view
5. Announcements
6. Push notifications
7. Guidance referral status
8. Intervention tracking summary

This document is project context, not an instruction to implement all modules at once.
