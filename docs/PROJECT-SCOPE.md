# LENS Initial Release Scope

## Objective

Prepare Laravel for RFID attendance and build a Flutter Android application where parents or guardians can log in, view linked children, receive attendance notifications, and read school announcements.

## Initial Users

- System Administrator
- School Administrator
- Parent or Guardian
- RFID Device

## Included

### Laravel

- administrator authentication
- school configuration
- student records
- guardian records
- student-guardian links
- RFID device registry
- RFID card assignment
- scan ingestion
- attendance processing
- announcements
- notification records
- parent mobile APIs
- audit logging

### Flutter

- parent login
- secure remembered session
- linked children
- attendance history
- announcements
- notification inbox
- push notification handling
- profile and logout

## Explicitly Excluded

- teacher app
- student login
- grading and report cards
- LMS
- payments
- SMS
- guidance case management
- facial recognition
- QR attendance
- transport
- booking
- inventory
- AI
- iOS
- multi-school tenancy
