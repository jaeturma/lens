# LENS Minimal Claude Code Workspace

This package adds a lightweight Claude Code development environment to an existing Laravel 13 project with a Flutter Android client.

Start with `STEP-BY-STEP.md`.

Included:

- concise global project rules
- Laravel, Flutter, and testing rules
- four reusable Claude Code skills
- project and architecture documents
- work-package template
- three starter work packages
- step-by-step invocation instructions

The package does not modify Laravel application source code, install dependencies, or assume a particular Flutter state-management package.
